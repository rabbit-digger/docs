# 介绍

rabbit-digger 是由 Rust (opens new window) 编写的代理软件.

目前还处于快速开发阶段. 文档可能与实际用法不一致, 如果有发现不一致请 提交issue (opens new window).

# 支持的代理协议

  • Shadowsocks
  • Trojan
  • HTTP
  • Socks5
  • obfs(http_simple)

# 支持的服务器协议

  • Socks5
  • HTTP
  • http+socks on the same port
  • Shadowsocks
